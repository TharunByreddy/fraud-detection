# Fraud Detection
End-to-end fraud detection with XGBoost, LightGBM, SMOTE, and SHAP explainability.

## Stack
XGBoost · LightGBM · imbalanced-learn · SHAP · Streamlit · Plotly

## Results
| Model | ROC-AUC | PR-AUC | F1 |
|---|---|---|---|
| XGBoost | 0.9724 | 0.8813 | 0.8342 |
| LightGBM | 0.9681 | 0.8654 | 0.8198 |
| Random Forest | 0.9512 | 0.8201 | 0.7923 |
